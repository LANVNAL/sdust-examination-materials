# 山东科技大学考试资料
## 1. 这是什么
仓库收录山东科技大学的历年考试相关资料，目前以计算机学院的资料为主
## 2. 我能随意获取吗
仓库中的资料是共享性质的，任何人可以随意获取。
另外资料如有**侵权**，请私聊仓库的维护人员
## 3. 其他人如何贡献资料
若想为此仓库提交资料，请克隆本仓库后提Pull Request。
